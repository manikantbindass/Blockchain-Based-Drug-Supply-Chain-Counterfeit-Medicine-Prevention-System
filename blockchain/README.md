# Blockchain Layer

Hyperledger Fabric network for immutable drug supply chain ledger.

## Purpose

Provides decentralized, tamper-proof transaction ledger for batch tracking.

## Components

- fabric-network/: Network configuration and setup
- chaincode/: Smart contracts (Go)

## Technology

- Hyperledger Fabric 2.5+
- Docker containerization

## How This Fits

Core layer ensuring data immutability and distributed consensus.
